# Portfólio Pessoal

Este projeto é um portfólio pessoal construído com React, Vite e Tailwind CSS, com foco em uma experiência visual elegante, moderna e responsiva. A interface foi pensada para apresentar informações de forma limpa, destacando projetos, formação e contato.

## Demonstração

A versão online está disponível em:

- https://maiolli.vercel.app/

## Recursos

- Design escuro e minimalista
- Página inicial com apresentação pessoal
- Seção de sobre mim
- Página de projetos com destaque para trabalhos relevantes
- Página de contato
- Navegação fluida e responsiva

## Tecnologias utilizadas

- React
- Vite
- Tailwind CSS
- React Router
- Lucide React

## Estrutura do projeto

```bash
src/
├── components/   # Componentes reutilizáveis
├── data/         # Dados do perfil e dos projetos
├── hooks/        # Hooks personalizados
├── pages/        # Páginas da aplicação
├── App.jsx       # Definição das rotas
├── main.jsx      # Ponto de entrada
└── index.css     # Estilos globais
```

## Como executar localmente

Pré-requisitos:

- Node.js 18 ou superior

```bash
npm install
npm run dev
```

O projeto ficará disponível em http://localhost:5173.

## Build para produção

```bash
npm run build
```
